Visitor Counter

For my college mini project 1, I created a visitor counter using basic electronic components. IR sensors detect movements, and a 555 Timer IC debounces the signals. A 74LS90 counter IC tracks the visitor count, which is displayed on a 7-segment display. This project highlights my skills in practical electronics and circuit design.

Water Flow Meter

For my college mini project 2, I developed a water flow meter using an Arduino. The system measures water flow through a sensor connected to the Arduino, which processes the data to calculate the flow rate. The results are displayed on an LCD screen.
